## Overview
Connector-ts provides open source NestJs application that allows you to access the Boomtown SDK from your application.

## Technologies
- NestJs

## Prerequisites
Please make sure that Node.js (>= 6.11.0) is installed on your operating system.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
